# SEO Quick Start Guide - Yannova.be

## 🚀 Start Vandaag (15 minuten)

### 1. Google Search Console Verificatie
```bash
# Ga naar: https://search.google.com/search-console
# Voeg toe: https://www.yannova.be
# Kies verificatiemethode en volg instructies
```

### 2. Sitemap Indienen
```
Sitemap URL: https://www.yannova.be/sitemap.xml
```
- Dien in bij Google Search Console
- Dien in bij Bing Webmaster Tools

### 3. Google Business Profile
```bash
# Ga naar: https://business.google.com
# Claim je bedrijfsprofiel
# Vul alle gegevens in
```

## 📊 Wat is al Klaar?

✅ Sitemap gegenereerd en geoptimaliseerd
✅ Robots.txt geconfigureerd  
✅ Structured data (Schema.org)
✅ Meta tags op alle pagina's
✅ Mobile-friendly design
✅ Fast loading times
✅ Google Analytics via GTM
✅ Microsoft Clarity tracking

## 🎯 Deze Week Doen

1. **Google Search Console** - Verificatie + sitemap indienen
2. **Google Business Profile** - Claimen en invullen
3. **Reviews** - Vraag 5 klanten om Google review
4. **Foto's** - Upload 10+ projectfoto's naar Google Business

## 📈 Volgende Maand

1. **Content** - 4-8 blogposts schrijven
2. **Backlinks** - 10+ links opbouwen
3. **Social Media** - Facebook + Instagram opstarten
4. **Monitoring** - Wekelijks rankings checken
